# Research and Academic Literature

## Papers with Code
https://paperswithcode.com/  
"A free and open resource linking machine learning and computer science research papers with the actual code needed to implement them."

## Connected Papers
https://www.connectedpapers.com/  
"A visual tool to help researchers and applied scientists find, explore, and map out academic papers and citation graphs."

## arXiv – Computer Science
https://arxiv.org/archive/cs  
"The standard open-access archive for millions of pre-print scholarly articles in computer science, mathematics, and related fields."
