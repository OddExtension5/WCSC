# Game Development

## Godot Engine
https://godotengine.org/    
"A completely free and open-source game engine ideal for both 2D and 3D games, featuring a highly intuitive node-based architecture and supportive community."

## Pygame
https://www.pygame.org/  
"A cross-platform set of Python modules designed for writing video games. A fantastic starting point if you already know Python and want to build interactive visual projects."

## Microsoft MakeCode Arcade
https://arcade.makecode.com/  
"A web-based, beginner-friendly platform for creating retro arcade games using visual blocks, Python, or JavaScript—perfect for introducing younger learners to game logic without needing high-end hardware."

## Unity Learn
https://learn.unity.com/  
"Comprehensive, project-based tutorials ranging from absolute beginner scripting to advanced 3D game physics and graphics rendering."
